# Treasure Hunt App - Screenshots

This folder contains screenshots of the Treasure Hunt App interface and features.

## App Screens

### Home Screen
Initial screen displaying the 5 treasure hunt targets:
- Leaf (nature object)
- Door (architecture object)
- Tree (nature object)
- Water (natural element)
- Animal (wildlife)

Each target shows its name, icon, and a search button.

### Camera Screen
Allows users to:
- Take photos of found objects
- Record short videos (max 10 seconds)
- Shows the current target being searched
- Displays camera permission request if needed

### Found Items List
Displays all successfully found items:
- Organized by date reported
- Shows thumbnail for each item
- Indicates whether it's a photo or video
- Delete button to remove items

### Scoreboard Screen
Shows progress statistics:
- Total items found
- Completion percentage
- Motivational message
- Bar chart visualization of progress

### Splash Screen
Loading/welcome screen shown when app starts.

## Features Shown

1. Responsive UI design
2. Icon-based interface (Ionicons)
3. Camera integration for photo/video capture
4. AI-powered object detection results
5. Local data persistence
6. Clean and organized data presentation
